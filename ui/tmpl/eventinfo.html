Event Help

The events page lists the events that have been created by club members. Events have features, such as weather or not they require money, or if they are invite only. Any user can create an event and add or remove members. Members of an event will recieve event messages, and can also send messages to the other event members. Below is a list of event attributes and how the function on the site.

Name

This is the event name which will show up messages setnt to members. The event name can only use letters, numbers, and spaces - so be sure to leave out any special characters when creating an event.

Date

The event date and time can be set with this field. Some events will not have set dates, or may be TBD, so this can be left blank. Events without dates will not be automatically deleted. However, events that are more than two weeks old will be removed from the list.

Description

A short description of the event. This field does not have the same character limitations as the name, so you can be more creative :)

Owner

The owner of an event defaults to whomever is creating the event. However, this can be changed to anyone in the player list. Owners can add/remove members, mark members as paid, and update information like the date, etc.

Paid Event

If an event requires money this box should be checked. It will allow the event owner to mark members as paid for tracking purposes.

Cost

For paid events, you can enter the cost of the event in this field.

Invite Only
If an event is created as Invite Only it will not be visable to non-members. In this case only the event owner, Administrators, or users with the Calendar role can add members. Events can start as invite only, and be later changed to public.

Members
The members section shows the current members and allows the owner to add, remove, or mark a member as paid.
Messages
The messages section will show a list of the messages sent for the event and allow members to send new messages.
