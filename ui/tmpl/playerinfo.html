Player Help

Each person in the roster has a set of attributes that is used for various purposes by the site and by others. Below is a description of the current attributes that are saved, and some instruction on how to manage your own attributes.

Name

This is the user's first name and last initial. In case of any data breaches, we want to refrain from using last names when possible.

Example: Eric R

Preferred Name

This is a user's nickname. Most features use nicknames as that is typically the name that is most familiar to the group.

Example: Cowboy

Phone Number

The user's phone number is their cell phone number. The number must accept text messages from our official phone number to use the site. NOTE: The official phone number is 1-833-754-1670. Make sure not to block messages from this number!

Example: 650-555-1234

Email

Currenly the email field is optional. In the future there may be features added that could use email instead of text messages for communication.

Example: myemail@gmail.com

GHIN Number

A user's GHIN number is also optional at the moment. Having the GHIN number will be useful in the future for tracking handicaps and making scorecards, etc. for 18 hole tournaments.

Example: 1234567

Managing Your Profile

Individuals can maange their data by clicking on the sttings icon on the top right of the site and selecting "Edit Profile"

User Roles

There are several roles that are used to manage permissions throughout the site.

User
All users are given this role which allows global text messages to be sent to them and general access to view the site. There are a few users that may only want specific event messages, and may not have this role.
Administrator
Administrators have access to add, delete, and modify users and events - including a user's role assignments.
Communications
All users should have the Communications role. This role allows a user to send messages either through the global messaging page or specific event pages. Administrators can revoke this access from a user if necessary, so play nice!
Roster
Members with the Roster role can add, delete, and modify users. Having this role does not allow for changing a user's roles.
Calendar
Members with the Calendar role can modify and delete events that they are not the owner of. Owners of an event can always modify event details or delete their event.
GHIN
The GHIN role is not currently used. It's future purpose will be for managing handicaps and helping with the NCGA subscription process for our club.
